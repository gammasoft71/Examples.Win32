# Generate and build

To build this project, open "Console" and type following lines:

````
mkdir build && cd build
cmake .. 
start HelloWorldTest.sln
```

Open "Test/Test Explorer" with Ctrl+E,t and type Ctrl+R,v to build and run all tests.
